Internal Variable
1. echo $BASH (path to binary)
2. echo $BASHPID | echo $$ (current PID)
3. echo $BASH_VERSION (The version of Bash installed on the system)
4. echo $EDITOR 
5. $EUID ("effective" user ID number) >> diff with $UID
